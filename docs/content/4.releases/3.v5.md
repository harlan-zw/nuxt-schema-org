## :icon{name="i-noto-warning"} Breaking Features

**Nuxt v3.16 required.** To avoid hoisting issues with the new Unhead v2, Nuxt Schema.org v5 requires Nuxt v3.16.

Please upgrade your Nuxt to continue using Schema.org.

```bash
nuxi upgrade --force
```
